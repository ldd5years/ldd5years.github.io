2018年6月11博客自动化问题

今天参考网上的教程 想要实现 远程在线管理 Hexo + 自动化部署（阿里云服务器自动同步Github代码）

理想实现： 移动端使用hexo admin 在线更新 Github代码 阿里云服务器 利用 fenkins 自动同步最新的Github代码

  • fenkins 自动部署 失败 参考资料（Github + Jenkins 实现自动化部署 hexo 博客静态文件）
  • hexo admin 无法远程使用 只能在本机 参考资料（通过 admin 后台管理 hexo 博客）

  1. fenkins 自动部署 失败

    可能是因为 shell命令设置

  2. hexo admin 无法远程使用 只能在本机

    暂时不知道，网上的教程没有理解透彻

    所以无法实现 远程管理

纠结：

如果hexo admin 必须部署在服务器上,那么fenkins的自动同步就是多此一举

思路：

  • 抛弃 fenkins + hexo admin 直接在hexo 项目部署在 服务器 由服务器提交代码到Github
  • 将hexo admin 安装在 服务器，抛弃 fenkins
